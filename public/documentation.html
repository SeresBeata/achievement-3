Movie API

The aim of this project is to build the server-side component of a “movie” web application. The purpose of the project to provide users with access to information about different movies, directors, and genres. As well as, to allow to users to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request body data format Response body data format
Get a list of all movies and their data /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[movieTitle] GET None A JSON object holding data about a single movie, containing a title, description, genre, releaseDate, director, imgPath and featured property.
Get a list of all movies and their data by genre /movies/moviebygenres/[genreName] GET None A JSON object holding data about all the movies with the specified genre.
Get a list of all movies and their data by director /movies/moviebydirectors/[directorName] GET None A JSON object holding data about all the movies with the specified director.
Return data about a genre by it's name. /movies/genres/[genreName] GET None A JSON object holding data about a single genre, containing a genreName and genreDescription property.
Return data about a director by name. /movies/directors/[directorName] GET None A JSON object holding data about a single director, containing a directorName, bio, birth and death property.
Allow new user to register. /users POST A JSON object holding data about the user to add. A JSON object holding data about the user that was added, including an id, username, password, email, birthday and favouriteMovies property.
Allow users to add a movie to favouriteMovies /users/[ID]/[movieID] POST None A JSON object holding data about the user, including an id, username, password, email, birthday and the updated favouriteMovies property.
Allow users to update their data. /users/[ID] PUT A JSON object holding data about the new data. A JSON object holding updated data about the user, including an id, username, password, email, birthday and favouriteMovies property.
Allow users to delete a movie from favouriteMovies. /users/[ID]/[movieID] DELETE None A JSON object holding data about the user, including an id, username, password, email, birthday and the updated favouriteMovies property.
Allow users to deregister. /users/[ID] DELETE None A text message indicating whether the user was successfully deregistered.
